# Commands

| Command | Purpose |
|--------|---------|
| `d0 add --local <dir>` | Install bundle |
| `d0 ls` | Installed bundles |
| `d0 <pkg>` | TUI browser |
| `d0 <pkg> ls` | Page slugs |
| `d0 <pkg> read <slug>` | One page |
| `d0 <pkg> search <q>` | Search |
| `d0 init --name @scope/n [dir]` | New bundle |
| `d0 build [dir]` | Package `.d0.tgz` |
| `d0 import <src> --name @scope/n` | Import markdown |
| `d0 mcp` | MCP stdio server |
