# Commands

| Command | Purpose |
|--------|---------|
| `doc0 add --local <dir>` | Install bundle |
| `doc0 ls` | Installed bundles |
| `doc0 <pkg>` | TUI browser |
| `doc0 <pkg> ls` | Page slugs |
| `doc0 <pkg> read <slug>` | One page |
| `doc0 <pkg> search <q>` | Search |
| `doc0 init --name @scope/n [dir]` | New bundle |
| `doc0 build [dir]` | Package `.d0.tgz` |
| `doc0 import <src> --name @scope/n` | Import markdown |
| `doc0 mcp` | MCP stdio server |
| `doc0 update [--check]` | Self-update the CLI from npm |
